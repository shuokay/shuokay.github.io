MXNet 源码详解 -- Static Graph

发表于 2020-10-18

在很多机器学习系统中，计算图是一个非常重要的概念，计算图是一个有向无环图，它描述的计算的依赖关系。有了 计算图之后，我们就可以根据static graph对计算做各种优化。

计算图

在计算系统的优化中，计算图可以帮助减少内存的占用和提升计算的并行度。
