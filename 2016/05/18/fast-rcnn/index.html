Multi-task Loss

дЄКйЭҐеЈ≤зїПиЃ≤еИ∞, Fast R-CNN зЪДдЄАдЄ™дЉШзВєжШѓ end-to-end зЪДиЃ≠зїГ, ињЩдЄАзЙєзВєжШѓйАЪињЗ Multi-task Loss жЭ•еЃЮзО∞зЪД.
зђђдЄАдЄ™ Loss жШѓзФ®жЭ•иЃ≠зїГbounding box зЪДз±їеИЂзЪД. иЊУеЗЇжШѓдЄАдЄ™з¶їжХ£зЪДж¶ВзОЗеИЖеЄГ, иЊУеЗЇзЪДиКВзВєдЄ™жХ∞жШѓ \(K+1\), \(p=\left(p_0,вА¶,p_k\right)\), еЕґдЄ≠\(K\) жХ∞жНЃйЫЖдЄ≠зЪДз±їеИЂжХ∞, \(1\) жШѓbackground
зђђдЇМдЄ™ Loss жШѓзФ®жЭ•иЃ≠зїГ bounding box regression offset зЪД, \(t^k=\left(t^k_x, t^k_y, t^k_w, t^k_h\right)\).
жѓПдЄАдЄ™ RoI йГљжЬЙдЄ§дЄ™ label, дЄАдЄ™жШѓз±їеИЂ \(u\), еП¶е§ЦдЄАдЄ™жШѓ bounding box regression target \(v\). Multi-task loss еЃЪдєЙдЄЇ:

$$ L\left(p,u,t^u,v\right)=L_{p,u}+\lambda\left[u \geq 1\right]L_{loc}\left(t^u,v\right) $$

еЕґдЄ≠, \(L_{cls}(p,u)=-\log{p_u}\) жШѓйТИеѓє classify зЪДloss
еП¶е§ЦдЄАйГ®еИЖ, \(L_{loc}\) жШѓеЃЪдєЙеЬ®дЄАдЄ™еЫЫеЕГзїДдЄКйЭҐзЪД bounding box зЪДжНЯе§±еЗљжХ∞. еѓєдЇОз±їеИЂ \(u\), еЕґground truth зЪДbounding box дЄЇ \(v=\left(v_x, v_y, v_w, v_h\right)\), еЕґpredict еЊЧеИ∞зЪДзїУжЮЬжШѓ \(t^u={t^u_x, t^u_y, t^u_w, t^u_h}\).
йТИеѓєbounding box regression зЪДloss еЃЪдєЙжШѓ:

$$ L_{loc}\left(t^u,v\right)=\sum_{i \in {x,y,w,h}}{\text{smooth}_{L_1}\left(t^u_i-v_i\right)} $$

еЬ®ињЩйЗМ,

\[ \text{smooth}_{L_1}\left(x\right) = \begin{cases} 0.5x^2 & \text{if } \lvert x \rvert < 1 \cr \lvert x \rvert -0.5 & \text{otherwise} \end{cases} \]
