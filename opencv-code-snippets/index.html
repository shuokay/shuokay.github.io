灰度拉伸

cv2.normalize(src[, dst[, alpha[, beta[, norm_type[, dtype[, mask]]]]]]) –> dst
设置 norm_type=cv2.NORM_MINMAX
alpha: 拉伸后的最小值
beta: 拉伸后的最大值
例如，拉伸到 0 到 255

像素值的线性变换

cv2.convertScaleAbs(src[, dst[, alpha[, beta]]]) –> dst

\[ g(x)=\alpha \cdot f(x) + \beta \]

计算之后，大于 255 的值被截断到 255
